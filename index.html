🔬 Data Science Projects

SA Crime Intelligence Report

A rigorous data science interrogation of South Africa's Q3 2025/26 crime statistics, cross-referenced with two decades of SAMRC femicide surveillance data. Applies correlation analysis, anomaly detection, OLS regression, and ARIMA forecasting to answer whether the reported 3.2% rape decrease reflects genuine progress — or fewer women trusting a broken system.

Python ARIMA Statsmodels Scikit-learn SAPS Data SAMRC
📓 View Notebook 📊 Dashboard

Mining Quality Intelligence

A full-stack ML system predicting silica concentrate quality failures in an iron ore flotation plant hours before they occur. Trained on 737,453 hourly sensor readings using XGBoost and a Neural Network with SMOTE balancing, business-optimal threshold tuning, and a live Streamlit dashboard with real-time GREEN / AMBER / RED operator alerts.

Python XGBoost TensorFlow Streamlit SMOTE Plotly
🚀 Live Dashboard GitHub

Tyla Grammy Sentiment Analysis

A powerhouse NLP study dissecting the global phenomenon of Tyla's historic Grammy triumph, revealing a staggering 82% positive sentiment across 214 authentic fan reactions spanning 20+ countries. Leverages dual-model sentiment validation (VADER + TextBlob) to separate genuine enthusiasm from algorithmic noise, with stunning interactive visualizations transforming raw YouTube commentary into actionable brand intelligence.

Python NLP VADER TextBlob YouTube API Pandas WordCloud
📓 View Notebook

Diamonds Are Forever

Full-stack ML study investigating the structural disruption of the global diamond industry — quantifying the 73.8% lab-grown price collapse since 2020, identifying five distinct buyer archetypes via K-Means clustering, and forecasting retail diamond prices with R²=0.9997 (Random Forest, RMSE=$136) — exceeding the published academic benchmark. Deployed as a live 4-page Streamlit intelligence dashboard.

Python Random Forest XGBoost K-Means Streamlit Plotly
🚀 Live Dashboard GitHub

SA Bank Trust Score

A data-driven consumer intelligence dashboard scoring South Africa's six major banks across complaint resolution, regulatory sanctions, consumer favour rate and public sentiment. Built with a weighted normalisation model and deployed as a live multi-page Streamlit app — helping South African consumers choose a bank based on evidence, not marketing.

Python Streamlit Pandas Matplotlib OBS Data SARB Data
🚀 Live Dashboard GitHub

Who Really Gets the Money?

A three-project public interest investigation into R69.5 billion of South African development finance. Combines inequality analysis (Gini, Lorenz curves), ML-powered cost-per-job prediction (Linear Regression, Random Forest, XGBoost), and unsupervised anomaly detection (Isolation Forest) to ask whether public DFI funding reaches the people who need it most — against a backdrop of 31.9% unemployment.

Python Scikit-learn XGBoost Isolation Forest Streamlit Plotly Civic Data
🚀 Live App GitHub
